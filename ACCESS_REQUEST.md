# Access Request Template

Use this template when requesting free early access to AI-Investor for non-commercial purposes.

## How to Submit

1. Go to [GitHub Discussions](https://github.com/noreplystandardprints-sketch/AI-Investor/discussions)
2. Click **"New Discussion"**
3. Select the **"Request Access"** category
4. Fill out the template below
5. Submit and await review (5-7 business days)

## Approval Criteria

Requests are approved based on:

✅ **Approved for:**
- Academic research and published papers
- Open-source projects (non-commercial)
- Educational use in classrooms or tutorials
- Personal development and learning
- Non-profit organizations
- Community contributions

❌ **Likely Rejected for:**
- Commercial trading operations (requires commercial license)
- Selling services/products using the code
- Hedging funds or profit-seeking use
- Resale of code or modifications
- Unclear or missing use case information
